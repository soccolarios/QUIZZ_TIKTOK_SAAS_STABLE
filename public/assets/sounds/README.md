# Sound Files for Quiz Overlay

Place the following sound files in this directory:

## Required Files

| Filename | Purpose | Recommended Duration |
|----------|---------|---------------------|
| `question_show.mp3` | Plays when a new question appears | 1-2 seconds |
| `countdown_warning.mp3` | Plays when timer reaches 5 seconds | 0.5-1 second |
| `correct_answer.mp3` | Plays when the correct answer is revealed | 1-2 seconds |
| `leaderboard_show.mp3` | Plays when leaderboard appears | 1-2 seconds |
| `next_question.mp3` | Plays during countdown to next question | 1-2 seconds |
| `tick.mp3` | Optional tick sound for countdown | 0.2-0.5 seconds |

## Sound Recommendations

- Use royalty-free sounds
- Keep volume moderate (not too loud)
- Short, clean sounds work best
- Avoid sounds with long tails/reverb
- Format: MP3, 128-192 kbps
